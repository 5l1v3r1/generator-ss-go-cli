> yeoman generator to develop cli tool with Golang

## Requirements

* Node.js
* [Yeoman](http://yeoman.io)

## Installation

```bash
npm install -g yo generator-ss-go-cli
```

Then generate your new project:

```bash
yo ss-go-cli
```
